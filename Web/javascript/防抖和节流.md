## 防抖函数

当一个动作连续触发，只执行最后一次。

### 应用

- 搜索框输入

- 正则验证

- 窗口大小

**简单实现：**

```js
function debounce(fn, delay) {
  let timer = 0
  return function () {
    if (timer) clearTimeout(timer)
    timer = setTimeout(() => {
      fn.apply(this, arguments)
      timer = 0
    }, delay)
  }
}
```

## 节流函数

限制一个函数在一定时间内只能执行一次

- 滚动加载

- 多次点击提交

- 搜索

**简单实现**

```js
const throttle = (func, wait) => {
    let timer;
    return () => {
        if (timer) {
            return
        }
        timer = setTimeout(() => {
            func();
            timer = null
        }, wait)
    }
}
```
