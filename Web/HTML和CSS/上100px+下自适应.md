## 多种方式实现上面 `100px` 下面自适应的布局

### Flex  实现

```html
<!DOCTYPE html>
<html lang="en">
  <head>
    <meta charset="UTF-8">
    <title>Flex 实现</title>
    <style>
      body,html{
        height: 100%;
      }
      .wrapper{
        height: 100%;
        display: flex;
        flex-direction: column;
      }
      .header {
        flex: 0 0 auto;
        height: 100px;
        background-color: rgb(103, 128, 87);
      }
      .content {
        flex:1;
        background-color: rgb(179, 40, 40);
      }
    </style>
  </head>
  <body>
    <div class="wrapper">
      <div class="header"></div>
      <div class="content"></div>
    </div>
  </body>
</html>
```

### 定位实现

```html
<!DOCTYPE html>
<html lang="en">
  <head>
    <meta charset="UTF-8">
    <title>position 实现</title>
    <style>
      body,
      html {
        height: 100%;
      }
      .wrapper {
        height: 100%;
        position: relative;
      }
      .header {
        position: absolute;
        top: 0;
        height: 100px;
        width: 100%;
        background-color: rgb(103, 128, 87);
      }
      .content {
        position: absolute;
        width: 100%;
        top:100px;
        bottom: 0;
        background-color: rgb(179, 40, 40);
      }
    </style>
  </head>
  <body>
    <div class="wrapper">
      <div class="header"></div>
      <div class="content"></div>
    </div>
  </body>
</html>
```

### grid

```html
<!DOCTYPE html>
<html lang="en">
  <head>
    <meta charset="UTF-8">
    <title>grid 实现</title>
    <style>
      body,
      html {
        height: 100%;
      }
      .wrapper {
        display: grid;
        height: 100%;
        grid-template-rows: 100px auto;
      }
      .header {
        background-color: rgb(103, 128, 87);
      }
      .content {
        background-color: rgb(179, 40, 40);
      }
    </style>
  </head>
  <body>
    <div class="wrapper">
      <div class="header"></div>
      <div class="content"></div>
    </div>
  </body>
</html>
```
